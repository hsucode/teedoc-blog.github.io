

# 创建投影

![](创建投影\2022-10-01-23-12-14.png)

选择投影命令，鼠标光标滑动至视图左、右、上、下侧，可生成不同方向的投影。
生成出的左右侧投影视图，只能在左右方向移动；上下侧投影视图只能沿着上下方向移动。

![](创建投影\2022-10-01-23-12-23.png)

![](创建投影\2022-10-01-23-12-34.png)
