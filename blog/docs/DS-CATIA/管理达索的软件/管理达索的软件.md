# 管理达索的软件

软件安装包分文件夹放置在不同的位置，方便以后安装

## 普通用户

对于一般用户，只需要以下的软件即可，文件夹也按照如下管理。__存储路径不要有中文__

    1. 3DEXPERIENCE_R2022x_Golden\3DEXPERIENCE_Native_Apps
    2. V6R2022x.FP.CFA.xxxx
