
# 常用软件集合


## 管理软件

由于每个app 版本不同，系统不同等因素，希望将同系列app 放在一个文件夹内管理，可以通过以下命令让每个app在自己的文件夹内。

mvapp.bat

```batch
@echo off
title mvapp
echo Please type you app name:
set /p TypeAppName=
md %TypeAppName%
move *%TypeAppName%* %TypeAppName%
```
